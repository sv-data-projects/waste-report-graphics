SV Annual waste report graphics index

This page contains links to a all data graphics developed for the 2019/20 versions of the Victorian Local Government Annual Waste Services Report and Victorian Recycling Industry Annual Report.

Only graphics used in the 'final' report outputs are 'finished': final copy-edited prose, visual layout and design, finish art, accessibility design (visual/palettes and screen reader), responsive design (for standalone embeddable formats), animations and transitions optimisation, interaction design and code documentation; are only 'finalised' where a graphic is used as a final output. Refer to the final outputs and/or contact the authors re: which graphics are 'final' and how they can be used for other purposes.

All graphics were originally designed as static assets for use in printed reports. However all graphics are 'programmed' and digital-first, meaning that are connected to data and be easily extended with more web-interactive features.

The linked dataset supporting these visualisations can be found and updated here. The code repository (on Github) for these visualisations can be found here (note: this will be moved to a SV data team repo shortly). The aim for each of these visualisations is that they can be reused with updated datasets and support more interactive and animated features where required. This is the rationale for rendering these visualisation directly from data and in the browser.

Most graphics are made in SVG, however table-like and text-heavy graphics use standard HTML elements. Accordingly, the suggested method for exporting is to use the web browsers print/save to PDF function to export a high quality vector graphic. Chrome is recommended (and tested), with settings for minimum or no margins or headers/footers. Some (HTML-based) graphics may require the 'background graphics' option to be checked to render as displayed on screen (i.e. where background CSS is used)
