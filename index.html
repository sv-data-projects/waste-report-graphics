SV Annual report graphics index

This page contains links to SVG based data visualisation being drafted for the upcoming 2019/20 Victorian Local Government Annual Waste Services Report and Victorian Recycling Industry Annual report.

All graphics are in draft and have initially been designed for embedded in the print reports. However all graphics are 'programmed' and digital-first, meaning that are connected to data and be easily extended for web-based rendering and use.

The linked dataset supporting these visualisations can be found and updated here. The GitHub repository (i.e. code) for these visualisations and apps can be found here (note: this will be moved to a SV data team repo shortly). The aim for each of these visualisation is that they can be reused with updated datasets and supporting more interactive and animated features in the future. This is the rationale for rendering these visualisation directly from data and in the browser.

Most graphics are made in SVG, with a handful using HTML elements. Accordingly, the suggested method for exporting is to use the web browsers print/save to PDF function to export a high quality vector graphic. Chrome is recommended (and tested), with settings for minimum or no margins or headers/footers. Some (HTML-based) graphics may require the 'background graphics' option to be checked to render as displayed on screen (i.e. where background CSS is used)

No interactive has yet been attached to these graphics, however those listed with a ☯ symbol have a 'dark mode' that can be toggled by pressing the 'shift' key while viewing the graphic
